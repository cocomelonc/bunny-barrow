# Artwork

Author: **cocomelonc**

All scenery, soil tiles, tunnels, roots, stones, water, carrots, particles,
interface shapes, and the rabbit are original procedural drawings rendered at
runtime by `BunnyBurrowView.java`. No downloaded tileset or image pack is used.
The original project artwork is licensed with the source under the MIT License.

The only bundled visual dependency is the Nunito typeface. It remains under the
SIL Open Font License 1.1; the exact license text is preserved in
`third_party/nunito/OFL.txt`.
